  ## Description
  This app builds a back end e-commerce site. It takes a working Express.js API and configures it to use Sequelize to interact with a MySQL database
  ## Requirments
  dotenv 8.2.0
  express 4.17.1
  mysql 2.1.0
  sequelize 6.35.2
